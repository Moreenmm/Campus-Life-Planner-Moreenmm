=============================================
            CAMPUS LIFE PLANNER
=============================================
This is a simple web application designed to help students manage tasks, study, events, and track the time spent engaging in these activities. This web application utilises HTML, JavaScript, and CSS.

=============================================
            FEATURES
=============================================
      **Task Management**
Adding new tasks with a title, date for the task, duration in hours, tags, and status of the tasks, whether it is upcoming or itâ€™s already completed.
Editing existing records in the Records table.
Deleting records with a confirmation.

      **Dashboard**
Analyses the total records count.
Records the total hours tracked.
Displays the top and most used tags.
Has a weekly target tracking which shows how many hours you have remaining or by how many hours you have exceeded your target.
ARIA live regions for screen reader updates for accessibility.

      **Records**
It has a smart search that is regex-supported.
Error handling for invalid regex patterns.
The search is valid when you use titles, tags and date to look for a task in the records table.

    **Data Persistence**
The recorded data remains even after refreshing the page.
JSON export and import functionality.
The exported data is stored locally in the records.json file after exporting.

      **Accessibility**
Clear form labelling.
Aria live region.
The skip to content link.

      **Responsive design**
Flexible navigation layout that is easy to use.
Mobile first layout.

      **Project structure**
Has the index.html file, which contains the structure of the application.
Style.css file contains the styling and responsive layout of the web application.
The  Script.js has the whole application logic.
